# Agent Routing Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `target_wraith_id` routing to PyWraith so operators can control remote wraiths through a peer chain without direct C2 connection. CLI sends commands to nearest wraith; commands are auto-routed to target wraith via hop-by-hop broadcast.

**Architecture:** Protocol adds `target_wraith_id` to `WraithMessage` envelope. Rust dispatcher checks this field on receipt — if matches local wraith_id, executes locally; if in peer table, forwards via Yamux stream 0; otherwise broadcasts to all peers with dedup via `seen_message_ids`. Response returns via reverse path.

**Tech Stack:** Rust ( Tokio, prost), Python (IPython shell), Protobuf

### Task 3: Rust Dispatcher — Route by `target_wraith_id`

**Files:**
- Modify: `src/wraith/dispatcher.rs`

The dispatcher currently handles only local commands. We need to add a separate path for forwarded commands that arrive via Yamux.

- [ ] **Step 1: Add `route_message()` method to dispatcher**

```rust
impl MessageDispatcher {
    /// Route a message: check target_wraith_id, forward to peer or broadcast
    pub async fn route_message(
        &self,
        mut msg: WraithMessage,
        state: Arc<Mutex<WraithState>>,
    ) -> Option<WraithMessage> {
        let target = msg.target_wraith_id.clone();
        let local_id = state.lock().unwrap().wraith_id.clone();

        // If targeted to us, dispatch locally
        if target.is_empty() || target == local_id {
            return self.dispatch(msg, state).await;
        }

        // If targeted to a direct peer, forward to that peer
        {
            let peer = state.lock().unwrap().peer_table.get(&target).cloned();
            if let Some(peer) = peer {
                if peer.sender.send(msg).await.is_ok() {
                    return Some(MessageCodec::create_command_result(
                        "".to_string(),
                        "forwarded".to_string(),
                        "".to_string(),
                        0, 0, "".to_string(),
                    ));
                }
            }
        }

        // Broadcast to all peers (except already-seen check done in caller)
        let peers: Vec<_> = state.lock().unwrap().peer_table.values().cloned().collect();
        for peer in peers {
            let _ = peer.sender.send(msg.clone()).await;
        }

        Some(MessageCodec::create_command_result(
            "".to_string(),
            "broadcast".to_string(),
            "".to_string(),
            0, 0, "".to_string(),
        ))
    }
}
```

```python
def do_set_target(self, arg: str = ""):
    """%set_target [wraith_id|none] - Set/clear global routing target."""
    if not self.connected:
        print("Not connected. Use 'connect' first.")
        return

    arg = arg.strip()
    if not arg:
        current = self.client.get_target()
        if current:
            print(f"Current target: {current}")
        else:
            print("No target set")
        return

    if arg == "none":
        self.client.clear_target()
        print("Target cleared")
    else:
        self.client.set_target(arg)
        print(f"Target set to: {arg}")
```
